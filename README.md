# Investor Partners Website

Eine professionelle Investment-Repräsentationsseite für General Atlantic und Lotus Investment GmbH.

## 🚀 Technologien

- **Next.js 15** (App Router)
- **React 18**
- **CSS Modules** (klassisches CSS)
- **Zweisprachig** (DE/EN)
- **Responsive Design** mit Hamburger-Menü

## 📦 Installation

```bash
# 1. Repository klonen
git clone https://github.com/ihr-username/investor-website.git
cd investor-website

# 2. Abhängigkeiten installieren
npm install

# 3. Entwicklungsserver starten
npm run dev

# 4. Im Browser öffnen
http://localhost:3000/de
